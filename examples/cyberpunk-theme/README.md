# Cyberpunk Theme Example

This example showcases:

- Cyberpunk theme tokens
- Tilt + glow card interactions
- Shimmer button
- Glitch-first loading accents

## Starter snippet

```tsx
<KitApp>
  <PageTransition type="slide">
    <GlowBorder>
      <TiltCard>
        <ShimmerButton>Deploy</ShimmerButton>
      </TiltCard>
    </GlowBorder>
  </PageTransition>
</KitApp>
```
