# i18n Integration Guide

How to internationalize applications built with Infini Dev Kit.

---

## Theme Typography (Built-in)

`ThemeTypography` provides locale-specific font stacks for English, Chinese, and Japanese:

```typescript
interface ThemeTypography {
  en: { body: string; heading: string; mono: string };
  zh: { body: string; heading: string; mono: string };
  ja: { body: string; heading: string; mono: string };
  sizes: { xs: string; sm: string; md: string; lg: string; xl: string; xxl: string; display: string };
  weights: { light: number; normal: number; medium: number; semibold: number; bold: number };
  lineHeights: { tight: number; normal: number; relaxed: number };
}
```

Each theme defines appropriate fonts per locale. Access via theme spec:

```typescript
import { getThemeSpec } from "@infini-dev-kit/theme-core";

const spec = getThemeSpec("cyberpunk");
const jaHeading = spec.typography.ja.heading; // e.g. "M PLUS 1 Code, sans-serif"
const enBody = spec.typography.en.body;       // e.g. "Inter, sans-serif"
```

## react-i18next Integration

### Setup

```tsx
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';

i18n.use(initReactI18next).init({
  resources: {
    en: { translation: { /* ... */ } },
    zh: { translation: { /* ... */ } },
    ja: { translation: { /* ... */ } },
  },
  lng: 'en',
  fallbackLng: 'en',
});
```

### Connecting to Component Labels

Components with `labels` props accept `Partial<*Labels>` — provide only the keys you need to override:

```tsx
import { useTranslation } from 'react-i18next';
import { MediaGallery } from "@infini-dev-kit/react";

function GalleryPage({ items }) {
  const { t } = useTranslation();

  return (
    <MediaGallery
      items={items}
      labels={{
        noMedia: t('gallery.noMedia'),
        pause: t('gallery.pause'),
        resume: t('gallery.resume'),
        fullscreen: t('gallery.fullscreen'),
      }}
    />
  );
}
```

Unspecified keys fall back to built-in English defaults.

## Font Selection by Locale

To apply locale-specific fonts from the theme:

```tsx
import { getThemeSpec } from "@infini-dev-kit/theme-core";

function App({ locale }: { locale: "en" | "zh" | "ja" }) {
  const spec = getThemeSpec("default");
  const fonts = spec.typography[locale];

  return (
    <div style={{ fontFamily: fonts.body }}>
      <h1 style={{ fontFamily: fonts.heading }}>Title</h1>
      <code style={{ fontFamily: fonts.mono }}>code</code>
    </div>
  );
}
```
